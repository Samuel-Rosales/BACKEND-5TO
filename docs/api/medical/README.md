# Medical

Base URL: `/api/v1/medical`

Este módulo corresponde al **Módulo Clínico y Pacientes** del `schema.prisma`.

## Modelos (Prisma) documentados aquí

- `MedicalSpecialty`
- `Patient`
- `Doctor`
- `Consultation`
- `Prescription`

Nota: el schema incluye otros modelos clínicos (síntomas, diagnósticos, exámenes) que pueden aparecer como sub-recursos de `Consultation`.
